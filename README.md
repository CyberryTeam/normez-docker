# normez-docker
Docker image for [NormEZ](https://github.com/ronanboiteau/NormEZ)

## Usage

### Current directory

```shell script
docker run --rm -v "$PWD:/files" cyberryteam/normez-docker:latest
```

### Help

```shell script
docker run --rm -v "$PWD:/files" cyberryteam/normez-docker:latest -h
```
