## Getting started

### Prerequisites

1. [Install Docker](https://docs.docker.com/get-docker/)

### Installation

1. Clone the repository:

```shell script
git clone https://github.com/CyberryTeam/normez-docker.git
cd normez-docker
```

2. Build the image:

```shell script
docker build -t normez-docker .
```
